大数据挑战赛:机器学习之域名向量化

使用字符嵌入模型(Character Embedding)完成域名的向量化Character Embedding 方法将每个单独字符视为一个特征，根据这个规则对所有域名的特征进行提取、去重，结果作为所有域名具有的特征。每个特征都对应着一个索引，对于域名中的每个字符，都用对应的特征索引进行替代，从而得到域名的索引表示。随后将其输入Embedding网络层，输出的向量就是域名的向量表示。
